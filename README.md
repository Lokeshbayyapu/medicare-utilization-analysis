# Medicare Claims Cost & Utilization Analysis

## Overview
Analyzed 558,852 Medicare records from CMS Market Saturation and Utilization dataset, identifying $4.24 Trillion in payment patterns across 17 service types and all US states.

## Tools Used
- Python (Pandas, Matplotlib)
- Power BI
- Tableau Public

## Key Findings
- Independent Diagnostic Testing highest Medicare spend at $1.24 Trillion
- California, Texas, Florida top 3 states by Medicare spending
- 22+ Million providers across all US states analyzed
- Dual eligible high-risk population identified across all states
